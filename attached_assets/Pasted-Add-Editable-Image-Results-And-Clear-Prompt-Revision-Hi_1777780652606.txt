Suggested section title:
Edit This Image

Suggested actions:
- Revise Image Prompt
- Change Style
- Change Lighting
- Change Composition
- Generate Another Version
- Use This As Reference

Important:
Do not claim true pixel-level image editing unless the backend actually supports image editing.
If the current system only supports generating a new image from a revised prompt, call it:
Generate Another Version
or
Revise Image Prompt

Do not use misleading labels like:
Edit Pixels
Edit Image Directly
unless that actually exists.

Correct flow:
Generated Image
↓
Download / Generate Another / Copy Prompt
↓
Edit This Image
↓
Revise Prompt / Change Style / Generate Another Version

- Show the revised image prompt before generating again.
- Let the user review/edit the revised prompt.
- Then use the existing Generate Image flow.
- Do not create a new backend unless required.

What Changed

Old Version

Old Version:
[previous prompt]

New Version:
[current prompt]

Changes Made:

- Added output format
- Added constraints
- Made tone more direct
