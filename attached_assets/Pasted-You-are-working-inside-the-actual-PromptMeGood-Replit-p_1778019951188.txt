You are working inside the actual PromptMeGood Replit project.

Goal:
Integrate the Prompt Engine and Photography Suite into the new PromptMeGood workstation so they can each stand alone, but also work together through Master Link to create one premium Master Actionable Plan.

Core Principle:
Prompt Engine and Photography Suite must be independent first, linked when the user chooses.

Do not force every user to use both.
Do not make Photography Suite feel like random extra settings for the text prompt.
Do not make Prompt Engine dependent on Photography Suite.
Do not make Photography Suite dependent on Prompt Engine.

Each tool must generate value on its own.
Master Link is the optional synergy layer.

Required Behavior:

1. Prompt Engine Standalone
If the user only uses Prompt Engine:
- Generate a strong text prompt.
- Allow Run With AI.
- Allow Copy.
- Allow Refine.
- Allow Save to Vault.
- Allow Expert Mode to analyze it.

Example use:
User wants a business plan, sales email, content idea, YouTube script, product description, or strategy prompt.

Output:
- Detailed text prompt.
- Optional AI result if Run With AI is clicked.

2. Photography Suite Standalone
If the user only uses Photography Suite:
- Generate a strong image/visual prompt.
- Allow image prompt copy.
- Allow Generate Image Here if current functionality supports it.
- Allow Send To Midjourney if current functionality supports it.
- Allow Save This Combo if current functionality supports it.
- Allow Save to Vault/history if available.

Example use:
User wants product photography, cinematic portrait, thumbnail direction, lifestyle image, ad visual, or AI image prompt.

Output:
- Detailed image prompt.
- Technical visual brief.
- Optional image generation result.

3. Master Link On
If the user uses both Prompt Engine and Photography Suite and turns Master Link on:
- Combine both sets of data.
- Generate a Master Actionable Plan.
- The output must feel unified, not pasted together.
- The final AI Execution prompt should include both narrative intent and technical visual direction.

4. Master Link Off
If Master Link is off:
- Keep Prompt Engine and Photography Suite outputs separate.
- Do not confuse the user.
- Show that each tool can generate independently.

User-Facing Master Link Copy:

Master Link Off:
"Prompt Engine and Photography Suite are working separately."

Master Link On:
"Master Link is on. Prompt Engine and Photography Suite will combine into one professional AI blueprint."

Short tooltip:
"Link the Soul and Body of your prompt into one Master Actionable Plan."

Required Output Format For Master Actionable Plan:

## The Narrative
[Generated from Prompt Engine. Explain the idea, goal, audience, tone, message, story, and creative direction.]

## Technical Brief
[Generated from Photography Suite. Explain camera settings, lens, lighting, composition, film stock, color, mood, and visual execution.]

## AI Execution
[The final usable prompt string that combines narrative direction and technical direction. Include AI parameters when appropriate, such as --ar 16:9 --v 6.0.]

Example:

## The Narrative
Create a premium short-form video concept for a wearable fitness ring aimed at busy adults who want effortless health tracking. The tone should feel modern, confident, useful, and emotionally motivating. Focus on convenience, self-awareness, and daily performance without exaggerating medical claims.

## Technical Brief
Use a cinematic close-up of the ring on a hand gripping a gym bag. Shoot with an 85mm lens at f/1.8 for shallow depth of field. Use soft teal rim lighting, warm highlights, ISO 400, 1/125s shutter speed, and Kodak-inspired color treatment. Composition should feel premium, minimal, and tech-forward.

## AI Execution
Create a premium short-form video ad concept for a wearable fitness ring for busy adults who want effortless health tracking, modern confident tone, convenience and daily performance angle, cinematic close-up of ring on hand gripping gym bag, 85mm lens, f/1.8, shallow depth of field, soft teal rim light, warm highlights, ISO 400, 1/125s, Kodak-inspired color grade, premium minimal tech aesthetic --ar 16:9 --v 6.0
