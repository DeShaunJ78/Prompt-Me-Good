I re-audited PromptMeGood using the same lens as before—full-site flow, user friction, top-to-bottom clarity, button usefulness, and post-generation completion—and the site is better now. The product is clearer, more premium, and more complete than before, especially because the page now explicitly supports Fix My Prompt, local vault clarity, keyboard-first workflow, in-app AI use, and image download/regenerate/reference behavior.

Site score
Updated score: 8.8/10. The site now does a stronger job of telling users what PromptMeGood is, what it fixes, what features exist, and how to move from a vague idea to a usable prompt or generated image. It still falls short of 9+ because one major clarity issue remains: the homepage still says “Click Generate” in the “How It Works” section while the core product flow and Quick Start correctly say “Tap Fix My Prompt,” which creates avoidable friction in the most important interaction on the site.

What improved
Several upgrades are clearly visible now. The page now frames the product as “Finally, AI That Understands You,” explains why prompts fail, adds stronger feature packaging with Structured Prompt Builder, Photography Suite, DALL·E 3 Image Generation, Run With AI In-App, Local Prompt Vault, and Keyboard-First Workflow, and explicitly says users can generate, download, regenerate, and use images as a reference in the same tab. The Prompt Vault explanation is stronger too, because it now says the last 25 prompts are saved on the device and that users can export or import, which improves trust and product completeness.

Remaining friction
The biggest remaining issue is still label inconsistency. The page’s “How It Works” says “Describe what you need, Click Generate, Copy and use anywhere,” but the deeper product language, Structured Prompt Builder description, and Quick Start center the correct action around Fix My Prompt, so first-time users can still wonder whether “Generate” and “Fix My Prompt” are different actions. There is also still some density in the middle of the page: Help Me Start, This Week’s Focus, Your Fixed Prompt, Pinned Note, Refine Your Prompt, strength analysis, supported AI tools, input summary, best prompts, vault, Quick Start, and full guide all appear on one long page, which is powerful but still somewhat cognitively heavy for new users.

Buttons and actions
Most of the key actions now read as useful from a user standpoint:

Fix My Prompt now feels like the right product action and should be the universal text CTA.

Help Me Start is still a standout feature and is correctly positioned as a beginner-support tool.

Run With AI right here is strong because it closes the loop between prompt creation and prompt testing.

Copy Prompt remains essential and should stay highly visible.

Generate Image Here is appropriately separate from text prompt fixing and better matches user expectation for image output.

Download, Regenerate, or Use As Reference is now clearly described and is one of the biggest improvements in post-generation completion.

Prompt Vault is now more trustworthy because it explains storage and restoration more concretely.

The remaining opportunity is not feature coverage—it is action hierarchy. Once a prompt appears in Your Fixed Prompt, the page should still make one next step more visually obvious than the others, ideally Copy Prompt or Run With AI, so users do not pause to decide what the “main” action is.

How to get over 9
You are close. To move from 8.8 to 9.2+, I would focus on only four things:

Replace “Click Generate” in the top “How It Works” section with “Tap Fix My Prompt” so the homepage speaks with one voice.

Add one short confirmation line directly in the output state, such as “Your prompt is ready. Copy it, run it, or refine it.”

Make the post-result action order unmistakable, with one primary button and the rest secondary.

Slightly reduce middle-page cognitive load by grouping related utilities more tightly, especially analysis, input summary, best prompts, and vault, so the core path stands out faster.

The short version: you made real progress. PromptMeGood now feels much closer to a polished prompt-engineering product and less like a loose collection of AI tools, and with one more consistency pass plus stronger action hierarchy, it can reasonably break the 9/10 threshold.
