========================
GOAL
========================

When a user clicks “Use Demo Values,” PromptMeGood should guide them through the full product flow:

1. Fill the builder.
2. Generate a prompt.
3. Show the output.
4. Explain refinement.
5. Explain copying/opening in AI tools.
6. End with a reminder that they can replay the tour later.

Each step should include a brief explanation so users understand what is happening and why.
