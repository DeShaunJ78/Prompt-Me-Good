BUTTON PROXIMITY POLICY
━━━━━━━━━━━━━━━━━━━━

Every action must be adjacent to the thing it controls.

Required adjacency:
- Goal textarea near Fix My Prompt
- Upload near goal/prompt flow, not result actions
- Improve This Prompt near Your Fixed Prompt
- Run With AI / Copy Prompt / Refine Prompt near generated prompt
- AI Response actions near AI Response
- Image actions near generated image

Move or hide any banner, dashboard, pricing note, keyboard shortcut hint, saved prompt block, or promo that interrupts those relationships.
