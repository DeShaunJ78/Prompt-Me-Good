PromptMeGood is an AI prompt builder at promptmegood.com. 
Users type a goal, click Fix My Prompt, get a structured 
AI prompt, then run it with AI in-app or copy it to 
ChatGPT, Claude, or Perplexity. It also generates images 
via DALL-E 3 in Create An Image mode.

There are two complete user journeys:

JOURNEY 1 — PROMPT PRODUCTION:
Land → Type Goal → Fix My Prompt → See Generated Prompt 
→ Choose: Run With AI / Copy / Refine → Done

JOURNEY 2 — IMAGE PRODUCTION:
Land → Switch To Create An Image → Answer 6 Photo 
Questions → Build My Image Prompt → Generate Image 
→ See Image → Download → Done

Every element on the page must serve one of these 
two journeys or support them directly. Nothing else.
