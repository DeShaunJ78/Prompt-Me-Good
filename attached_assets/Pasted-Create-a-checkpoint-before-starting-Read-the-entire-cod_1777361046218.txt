━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

On mobile the keyboard triggers when tapping
non-input buttons. This destroys the experience.

1. After every non-input button click handler:
setTimeout(function() {
  if (document.activeElement &&
    document.activeElement.blur) {
    document.activeElement.blur();
  }
}, 10);

Apply to: #generateBtn, #image-generate-btn,
#runBtn, #copy-btn, all .btn-secondary,
all .btn-ghost, all .accent-swatch, all
.popular-use-card, all mode switch buttons,
all pill option buttons, all history buttons,
all template buttons, all accordion summaries,
all refine buttons.

2. Add inputmode="none" to every button that
is not a submit for a text form.

3. Sticky bottom bar input:
No autofocus. autocomplete="off"
autocorrect="off" autocapitalize="off"
spellcheck="false"

4. Wizard container: touch-action: pan-y
Next and Back: inputmode="none"

      var(--color-primary) 50%, transparent);
  }

@keyframes pmgShimmer {
  0% { background-position: -200% 0; }
  100% { background-position: 200% 0; }
}

  0% { transform: scaleY(1); }
  50% { transform: scaleY(0.96); }
  100% { transform: scaleY(1); }
