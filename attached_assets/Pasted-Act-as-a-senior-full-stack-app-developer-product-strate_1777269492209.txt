========================
NEW CORE FEATURE (CRITICAL)
========================

Users must be able to upload files and images and have AI analyze them intelligently.

SUPPORTED INPUT:
- Text input (existing)
- Uploaded files (PDF, DOCX, TXT)
- Images (JPG, PNG, screenshots)

USER EXPERIENCE:

1. User types a goal:
   “Summarize this contract”
   “Explain this report”
   “What’s wrong with this screenshot?”

2. User optionally uploads:
   - a document OR
   - an image OR
   - both

3. PromptMeGood:
   - analyzes the file/image
   - builds a powerful prompt automatically
   - sends it to AI
   - returns a clear, useful response

UI REQUIREMENTS:

- Add an upload area near the goal field:
  “Upload a file or image (optional)”

- Show:
  - file name
  - preview for images
  - remove button

- Do NOT clutter UI
- Keep it simple and optional

BACKEND REQUIREMENTS:

- Add endpoint:
  POST /api/analyze

- Accept:
  - text input
  - file upload (multipart/form-data)

- Process:
  - extract text from documents
  - send image + prompt to vision-capable model
  - combine user goal + extracted content into structured prompt

- Return:
  {
    success: true,
    prompt: "...optimized prompt...",
    response: "...AI answer..."
  }

SECURITY + COST:

- File size limits (ex: 5MB–10MB)
- Validate file types
- Rate limit requests
- Do NOT store files permanently unless needed
- Keep API usage controlled
