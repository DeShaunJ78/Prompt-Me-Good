PromptMeGood Expert Command Center Label Update

Goal:
Add clear labels, helper text, and short explanations to the Expert Command Center so users understand what each feature does.

Important:
Do not rebuild the Expert Command Center.
Do not change database logic.
Do not change API routes.
Do not change how features work.
Do not move Expert Mode into the beginner flow.
Do not clutter the main homepage.
Keep Expert Mode hidden behind the existing Expert Mode button.
Preserve the current warning before entry.

This is a targeted copy and UX clarity update only.

Main Rule:
Every Expert Command Center feature should answer:
1. What is this?
2. Why would I use it?
3. What happens when I click it?

Use short, plain-language labels. Avoid prompt-engineering jargon unless it is explained.

UX Requirements:
- Labels should be visible without overwhelming the page.
- Helper text should be short.
- Use tooltips, small subtext, info icons, or expandable help where appropriate.
- Do not make the Expert Command Center feel crowded.
- Do not add long paragraphs.
- Keep the interface clean and premium.
- Use Title Case for short UI labels if that matches the current app style.
- Keep body helper text natural and readable.

Do Not Touch:
- Main beginner prompt flow.
- Fix My Prompt behavior.
- Help Me Start behavior.
- Run With AI behavior.
- Prompt Strength Meter calculation.
- Prompt Vault or saved work logic.
- Database schema.
- API routes.
- Pricing.
- Homepage layout outside Expert Mode.
