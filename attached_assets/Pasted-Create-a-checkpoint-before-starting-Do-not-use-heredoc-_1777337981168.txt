Add a live prompt strength indicator that updates as the user types in the goal field. This is separate from the existing post-generation prompt strength score.

1. Place the indicator directly below the goal textarea. Show it as a pill with a colored dot and label:

0-20 characters: grey dot — "Too Short"
21-60 characters: yellow dot — "Getting There"
61-120 characters: blue dot — "Good"
120+ characters OR contains power words: green dot — "Strong ✓"

Power words that trigger Strong: who, what, how, why, help me, I want, I need, my, for, because, so that, make sure, avoid, include, step by step, list, explain, create, write, build, find, give me, show me

2. Visible on both mobile and desktop at all times while typing.

3. Updates on every keystroke — no button press required.

4. When indicator reaches "Strong" animate it with a brief subtle pulse using CSS animation to reward the user.

5. The indicator is hidden before the user starts typing. It appears on first keystroke. It disappears after generation and reappears when the user types again.
