PART 1: REPLIT WIRING INSTRUCTIONS
------------------------------------

The templates section already exists in the DOM as #templates with a #templates-grid inside it.
The four existing template cards are there but the section is not visible on the main screen.

Do these three things:

1. ADD A TRIGGER BELOW THE TEXTAREA
   Directly below the #goal textarea, add a small link:
   "Need inspiration? Browse templates"
   When tapped, it scrolls to or reveals the #templates section.
   Style it as small muted text — same ghost style as the preview nudge.

2. WIRE EACH TEMPLATE CARD TO THE TEXTAREA
   Every .template-card button needs an onclick that:
   a) Sets document.getElementById('goal').value = [the template's goal text]
   b) Scrolls back up to the textarea
   c) Focuses the textarea so the user can edit it immediately
   The goal text for each template is defined in Part 2 below.

3. REPLACE THE 4 EXISTING TEMPLATES WITH THE 25 BELOW
   Delete the current 4 cards in #templates-grid and replace with
   the 25 cards defined in Part 2. Keep the same .template-card
   HTML structure that already exists.

- CATEGORY (used for .template-card-meta)
