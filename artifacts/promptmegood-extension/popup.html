Send to PromptMeGood

Right-click any text or image, then choose Send to PromptMeGood.

How it works

  • Text: select words on any page, right-click, choose Send to PromptMeGood. The web app opens in Text mode with your selection prefilled.
  • Images: right-click an image and choose Send image to PromptMeGood. The web app opens in Image mode with the image URL ready to use as a starting point.

Where to send

Defaults to the live site. Point at your dev preview when testing locally (for example a .replit.dev URL).
