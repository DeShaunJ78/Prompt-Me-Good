How to use PromptMeGood

What this tool does: PromptMeGood turns a rough idea into a clear, structured prompt you can paste into ChatGPT, Claude, Perplexity, Google, or any AI/search tool — so you stop getting weak, vague answers and start getting useful ones.

1

Type your idea in the goal box

Write what you actually want help with — in plain words. Don't worry about being polished; the app sharpens it for you.

Examples: "Make money online fast", "Write a TikTok hook about budgeting", "Plan my first dropshipping store"
2

Pick Your Experience Level

Tell the AI where you're starting from. Beginner gets simpler steps and definitions. Advanced skips the basics and goes deeper. Choosing the right level avoids answers that are too obvious or too over-your-head.

3

Choose a Tone

This shapes how the answer feels. Professional sounds polished and business-ready. Aggressive pushes for bold, decisive moves. Casual reads like a friend talking. Expert sounds like a specialist breaking things down.

4

Pick an Output Format

Decide how you want the answer laid out. Step-by-step gives you a clear walkthrough. List gives quick scannable points. Detailed breakdown gives a deeper, fuller explanation.

5

Turn on Human Voice Mode

Flip this on when you want the AI's answer to read smoothly, with varied sentences and a real-person voice. It adds natural phrasing and reduces obvious robotic patterns.

6

Turn on Clarity Boost Mode

Use this when you want the response organized and easy to act on — clear sections, defined terms, called-out assumptions, and steps you can actually follow.

7

Use Money Mode when you want results, not theory

Switch this on when your goal is execution and outcomes. The prompt will push the AI toward practical, results-driven actions instead of long, abstract advice.

8

Turn on Photo Prompt Mode for image tools

Flip this on when you want to generate prompts for AI image tools like Midjourney or DALL·E. It enhances the prompt with lighting, camera details, composition, and visual depth.

9

Choose an Output Language

Pick the language you want the AI to respond in — English, Spanish, Portuguese, Russian, French, or German. Only the AI’s response language changes; the app stays in English.

10

Click Generate Prompt

The app builds a full, structured prompt for you in seconds. It adds the role, the rules, the format, and the next steps the AI should follow.

11

Click Copy Prompt

One click copies the whole prompt to your clipboard. Paste it into ChatGPT, Claude, Perplexity, Google, or any AI/search tool and run it.

12

Use your Saved Prompts & History

Every prompt you generate is saved automatically. Open History to revisit, restore, rename, tag, favorite, or compare past prompts. Save the best ones as templates so you can reuse them with one click.

Pro Tips for Better Results

  • Be specific about your situation in Extra Details — budget, deadline, platform, audience. The more context, the sharper the answer.
  • Don't overthink the goal box. A messy, honest sentence beats a polished vague one.
  • If the first answer is close but not perfect, click More Detailed, More Aggressive, or Beginner Friendly to refine the current prompt directly from the output area.
  • If the first prompt is close but not perfect, use Fine-Tune Your Prompt instead of starting over.
  • Save winning prompts as templates so you never have to rebuild them.
  • Try the example chips ("Try one of these") if you're stuck — they're great starting points.
  • Use Photo Prompt Mode for creating high-quality image prompts for Midjourney, DALL·E, and other AI image tools.
  • Combine all boosts (Money, Human Voice, Clarity, Photo) for the most powerful results.

4. Refining The Output

Refinement Buttons

One-tap upgrades that update the current prompt in place — no re-typing.

  • More Detailed — adds depth, examples, structure.
  • More Aggressive — sharper, more action-focused.
  • Beginner Friendly — simpler explanations.

Fine-Tune Your Prompt

Tell PromptMeGood exactly what to change. Type something like "make it shorter", "add examples", or "focus on beginners", then tap Update Prompt. Form inputs stay as they were.

Manual Editing

Click anywhere inside the output box and type. An "Edited manually" label confirms your edits are active. Edits flow into Copy, history, and Use Your Prompt In.

Undo Last Change

Steps you back through every refinement and manual edit. Generating a new prompt or restoring one from history clears the undo stack.

5. Using Your Prompt

Copy Prompt

Use Your Prompt In

Opens ChatGPT, Claude, or Perplexity in a new tab so you can paste and run. Test across all three to see which gives the best answer.

Print / Save PDF

Send the current prompt to print or save it as a PDF for offline reference.

6. History & Export

Prompt History

Export & Import

Export Everything downloads a JSON backup. Import Backup restores it on any device. Use it before clearing your browser or switching devices.

7. Pro Tips

  • Be specific. "Help me write a tweet about coffee aimed at remote workers" beats "help me write a tweet."
  • Use Extra Details. Budget, deadline, audience, platform — every detail sharpens the answer.
  • Refine instead of restarting. The refinement buttons + Fine-Tune Your Prompt are usually faster than typing a new goal.
  • Try different Personality Modes. One of the quickest ways to change a result.
  • Edit before you save. Click inside the output box, polish the wording, then copy.
  • Test across platforms. ChatGPT, Claude, and Perplexity each respond differently.
  • Save winners. Favorite, tag, or note prompts you'll want to reuse.
  • Combine boosts. Money + Human Voice + Clarity often produces the strongest results.
  • Use the "Need ideas?" feature when you're stuck. It's the fastest way to discover useful prompts.

8. Common Mistakes

  • Being too vague. Add what, who, where, and by when.
  • Ignoring output format. List vs step-by-step changes how usable the answer is.
  • Skipping the refinement tools. The first prompt is rarely the best one.
  • Starting over instead of refining. Use the buttons, Fine-Tune, or Undo first.
  • Skipping Money Mode when it fits. If the goal is execution and income, leaving it off softens the answer.
  • Forgetting to save winners. Tag and favorite prompts you'll want again.
  • Getting stuck thinking of the perfect idea instead of using the idea generator to get started.
